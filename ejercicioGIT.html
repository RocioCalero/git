  1. ¿Que es un sistema de control de versiones?

    Un sistema de control de versiones es la gestión los cambios realizados sobre los elementos, de un repositorio o un proyecto.

  2. ¿Cual es la diferencia entre git y github?

    Git es como se ejecuta los repositorios de Github en la consola. Github es como una "biblioteca" donde se sube todos los proyectos que se crean.

  3. ¿Cómo creo una rama en git?

    Para crear una rama engit se usa el comando git branch

  4. ¿Cómo clono un repositorio?

    1. Cuando creas un repositorio puedes ver el clone, hay eliges ssh y copias
    2. Ahora en la consola, en la carpeta que lo vayas a guardar haces un git clone y pegas lo que has copiado

  5. ¿Cómo puedo bajar los ultimos cambios de un repositorio?

    Para bajar los ultimos cambios de un repositorio, tenemos que hacer un git clone
